FORECAST

FOrward REsearch on Clinical And Survival Trends

For analysts building decision-analytic models, FORECAST streamlines the process of fitting, validating, and visualizing parametric time-to-event models derived from published Kaplan–Meier survival curves. Use our app to accurately estimate event hazards and monitor their evolution over time.

Installation Instructions (Windows)

  1. Unzip the files to your local disk.

  2. Run the run.bat file (no need to have R installed).

  3. Wait for the program to load in your default browser.

Installation Instructions (Other Systems)

  1. Make sure you have R and RStudio installed.

    Download R Download RStudio

  2. Unzip the downloaded file to your local disk.

  3. Go to ProgramFiles, open app.r in RStudio, and click on RunApp in the top right corner of the RStudio window.
